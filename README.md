# 🍽️ Eatly - Food Delivery Website

**Учебный проект для оттачивания навыков HTML, CSS и JavaScript.** Современный веб-сайт доставки еды с адаптивной версткой и интерактивными компонентами.

## 🎯 Цель проекта

Тренировка и демонстрация навыков:
- **Чистая семантическая HTML-разметка**
- **Современный CSS с адаптивным дизайном**  
- **Интерактивные JavaScript-компоненты**
- **Работа с внешними библиотеками**

## 🛠 Технологии и реализация

### **HTML**
- Семантические теги (`<header>`, `<main>`, `<section>`, `<article>`)
- Доступность (`aria-label`, `aria-hidden`, `visually-hidden`)
- Чистая структура с BEM-методологией
- Валидная разметка

### **CSS**
- **Flexbox и CSS Grid** для сложных макетов
- **Адаптивный дизайн** (мобильные, планшеты, десктоп)
- **CSS-переменные** для единой стилизации
- **Медиа-запросы** с mobile-first подходом
- **CSS-анимации** для плавных переходов

### **JavaScript компоненты**
1. **Кастомный селектор** - стилизованный элемент `<select>` с доступностью
2. **Свайпер для отзывов** - на базе библиотеки **Swiper.js**
3. **Бургер-меню** для мобильной навигации
4. **Интерактивные карточки** с hover-эффектами

## ✨ Ключевые особенности

### **Кастомный селектор**
- Полная стилизация нативного `<select>`
- Поддержка клавиатурной навигации
- ARIA-атрибуты для доступности
- Плавные анимации открытия/закрытия

### **Свайпер отзывов**
- Интеграция **Swiper.js** с кастомизацией
- Адаптивное количество слайдов
- Кастомные точки пагинации
- Плавные переходы между слайдами

### **Адаптивная верстка**
- Mobile-first подход
- Гибкие изображения и контейнеры
- Условное отображение элементов
- Оптимизация для всех устройств

## 🎨 Дизайн-решения
- **Чистая цветовая палитра** с акцентными элементами
- **Консистентная типографика** с иерархией
- **Карточный дизайн** для контента
- **Минималистичный интерфейс** с акцентами

---

**Проект демонстрирует умение работать с современными веб-технологиями, создавать адаптивные интерфейсы и интегрировать сторонние библиотеки для улучшения пользовательского опыта.**
